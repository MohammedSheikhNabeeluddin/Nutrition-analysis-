# Nutrition Analysis App

A full-stack nutrition tracking and analysis application.

## Tech Stack
- **Backend:** Rust, Axum (Web Framework), SQLite (Database via SQLx)
- **Authentication:** JWT, Argon2
- **Frontend:** HTML, CSS, JavaScript

## Features
- User Authentication (Login / Register)
- Local data storage with SQLite

## Getting Started

### Prerequisites
- [Rust](https://www.rust-lang.org/tools/install)
- SQLite

### Running the application
1. Clone the repository
2. Run the backend server: `cargo run`
3. Access the frontend (usually served at localhost).
